# YAPS Nutrition — Site Web

Site officiel de l'application YAPS Nutrition.

## Stack
- HTML / CSS / React (CDN)
- Three.js (animation 3D Hero)
- Babel standalone (compilation JSX en client)

## Déploiement
Hébergé sur Cloudflare Pages.

## Pages
- `/` Landing
- `/about` À propos
- `/features` Fonctionnalités (avec dashboard utilisateur)
- `/pricing` Tarifs
- `/sciences` Sources scientifiques
- `/download` Téléchargements
- `/contact` Contact & FAQ
- `/privacy` Politique de confidentialité
- `/terms` Conditions d'utilisation
